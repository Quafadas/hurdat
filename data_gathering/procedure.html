Code - main

  1. A get request is going to get us some html with the raw text in hurdat format
  2. Drop the whitespace and html tags at the start and end of the file
  3. Identify lines, in which the first two characters are letters ... these are the "header" lines of a system
  4. Add a marker to each line, identifying the set of lines which are part of that hurriance
  5. Group the lines by the identifier in the previous step
  6. Assign the header / track to the data model, by writing functions which respect the formatting documnent
  7. Write to JSON
